# Job Tracker App

A simple web application to track job applications and their status.

## Features
- Add job applications
- Edit job details
- Delete job entries
- Filter jobs by status
- Data persistence using localStorage

## Tech Stack
- HTML
- CSS
- JavaScript
- Git & GitHub

## Concepts Used
- DOM Manipulation
- Event Handling
- Arrays and Objects
- localStorage
- CRUD Operations
